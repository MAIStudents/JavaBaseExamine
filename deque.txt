В языке программирования Java класс ArrayDeque представляет собой динамическую
структуру данных, которая реализует интерфейс Deque.
Класс `ArrayDeque` реализован с использованием массива

преим.
- массив для хранения элементов -> быстрые операции с добавлением и удалением элементов с концов.
- автоматически расширяет свой внутренний массив при необходимости
- Вставка/удаление элементов с концов осуществляется за время O(1)

недост.
- не могут быиь вставлены или удалены в произвольной позиции (основан на массиве)
(- доп. затраты на увеличение массива)??

Одним из главных преимуществ `ArrayDeque` является его спосоность эффективно вставлять и удалять элементы с обеих сторон
Рассмотрим несколько примеров, как это работает:

    Чтобы добавить элемент в конец, начало очереди, мы используем метод addLast(), addFirst()
   deque.addFirst(5); // Добавляем 5 в начало

   Для удаления элемента с конца, начала очереди применяется метод removeLast(), removeFirst()
   Integer firstElement = deque.removeFirst(); // Удаляем первый элемент
   System.out.println(firstElement); // 5

    метод `contains()`, для проврки содержится ли элемент в очереди.

        ArrayDeque<Integer> deque = new ArrayDeque<>();
        deque.addLast(20);
        
        System.out.println(deque.contains(20)); // true
        System.out.println(deque.contains(40)); // false

полезен для поиска в ширину, но имеет поиск О(n)
