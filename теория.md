- ### Интерфейс Set. Основные реализации. Как работают вставка, удаление, поиск элемента на примере одной из реализаций.
- Set - коллекция без повторения элементов. Знакомые мне реализации: TreeSet (вставка Log(n)), HashSet (вставка O(1)). Какая-либо упорядоченность в HashSet не гарантируется.
- Разберём Set на примере HashSet:
```java
import java.util.HashSet;

public class SetTest {
    public static void main(String[] args) {
        HashSet<String> set = new HashSet<>();
        set.add("apple");
        set.add("banana");
        set.add("cherry");
        set.add("apple");
        System.out.println(set);

        set.remove("apple");
        System.out.println(set);

        System.out.println(set.contains("cherry"));
        System.out.println(set.contains("chery"));

    }
}
```


- ### Класс ArrayList, LinkedList. Основные реализации. Как работают вставка, удаление, поиск элемента на примере реализаций.
- List - динамически расшираемый массив. ArrayList - имеет сплошное представление в памяти (быстрый доступ). LinkedList - разбросан "узлами" по памяти (медленный доступ, O(1) вставка).
- Разберём List на примере:
```java
import java.util.ArrayList;
import java.util.List;
```
```java
// List<Integer> intList = new LinkedList<>();
List<Integer> intList = new ArrayList<>();

intList.add(2);
intList.addFirst(3);
intList.addLast(1);
intList.add(0, 0); // добавление 0 вперёд
intList.add(2, 15); // добавление 15 третьим

intList.remove(0);
System.out.println("index = " + intList.indexOf(15));
System.out.println(intList.toString());

for (int i = 0; i < intList.size(); ++i)
{
    System.out.println(intList.get(i));
}
```
